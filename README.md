# MovilStore

Tienda web académica de smartphones y accesorios, desarrollada con HTML, CSS y JavaScript.

## Funciones

- Catálogo de smartphones y audífonos.
- Sección de ofertas y filtros.
- Carrito de compras.
- Pago simulado con Yape, Plin o tarjeta.
- Registro, inicio de sesión e historial de pedidos local.
- Diseño responsive y futurista.

> Proyecto demostrativo: no procesa pagos reales y guarda los datos únicamente en el navegador.
